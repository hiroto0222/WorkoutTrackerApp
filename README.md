# WorkoutTrackerApp

### Tech Stack
backend:
- Golang, Gin
- Postgresql, Gorm
- Firebase authentication

client:
- TypeScript, React Native + Expo
- Redux

### Local Deployment
client-android:
- npx react-native run-android / yarn android
- in case of crashes, run ./gradlew clean in ./android
